# Retro Snake: C-Based Logic Engine 🐍🎮

A high-performance implementation of the classic Snake game, built in C to demonstrate efficient memory management and real-time game loop logic.

## 🚀 Technical Highlights
* **Data Structures:** Uses a **Singly Linked List** (or dynamic array) to represent the snake's body, where each node stores (x, y) coordinates.
* **Game Loop Architecture:** Implements a non-blocking input system using `conio.h` (on Windows) to maintain a smooth frame rate.
* **Collision Logic:** Optimized algorithms for boundary detection and self-collision checks.

## 🛠️ Tech Stack
* **Language:** C
* **Concepts:** Pointers, Structs, Memory Allocation (`malloc`), and Console Manipulation.
* **Compiler:** GCC (MinGW for Windows).
